# Trader Behavior vs Market Sentiment Analysis

## Overview
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using historical trading data.

The objective is to identify behavioral patterns in trader profitability, risk-taking, and trade direction under different sentiment conditions.

## Dataset
- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trader Data

(Note: datasets are not uploaded due to GitHub file size limits)

## Tools Used
Python  
Pandas  
Matplotlib  
Seaborn  

## Analysis Performed
1. Data cleaning and preprocessing
2. Merging sentiment and trading datasets
3. Trader profitability vs sentiment
4. Long vs short trading behavior
5. Risk analysis using position size

## Key Insights
• Trader profitability varies across Fear and Greed periods  
• Traders tend to take larger positions during Greed markets  
• Market volatility during Fear causes larger PnL swings  

## Conclusion
Understanding trader behavior during different sentiment phases can help build smarter trading strategies and better risk management systems.
